What is JavaScript? ▶
JavaScript is a programming language used to make web pages interactive.
What are the primitive data types available in JavaScript? ▶
Primitive types include String, Number, Boolean, Undefined, Null, Symbol, and BigInt
What is DOM manipulation? ▶
DOM manipulation in JavaScript is the process of using the Document Object Model (DOM) API to dynamically interact with and modify the content, structure, and style of a web page after it has loaded.
For what javascript is used ▶
javascript is use to create dynamically updating content, control multimedia, animate images, and implement complex features across various platforms
